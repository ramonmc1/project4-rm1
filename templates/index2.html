Supervised Machine Learning - "Linear Regression"

Linear regression belongs to the category of supervised machine learning algorithms. The model is trained on a set of labeled data (training data) and then use the model to predict labels on unlabeled data (testing data).

Removing Outliers Results: By removing outliers, our linear model score improved from 0.5 to 0.63.
